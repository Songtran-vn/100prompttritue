# 💎 Diamond Prompt

> 100 câu lệnh AI trải nghiệm đỉnh cao — Trần Đức Sông · 2026

Ứng dụng học và ôn luyện 100 câu lệnh (prompt) ChatGPT được chọn lọc từ hơn 1.000 câu lệnh thực chiến.

## Tính năng

- 📖 **Học lần lượt** — xem từng câu lệnh, đánh dấu *Đã hiểu* hoặc *Muốn xem lại*
- 🔖 **Danh sách xem lại** — tổng hợp các câu cần ôn, có thể in ra PDF
- 🧠 **Kiểm tra** — hiển thị nửa câu lệnh, tự điền phần còn lại rồi xem đáp án
- 💾 Tự động lưu tiến trình vào trình duyệt (localStorage)

## Deploy lên Vercel

1. Fork hoặc clone repo này về GitHub của bạn
2. Vào [vercel.com](https://vercel.com) → **Add New Project**
3. Import repo từ GitHub
4. Vercel tự nhận diện static site — nhấn **Deploy**
5. Xong! App sẽ có URL dạng `https://diamond-prompt.vercel.app`
